3. ACTION POTENTIAL DURATION (APD)
================================================================================

APD90 (time to 90% repolarization):
------------------------------------
Human ventricular myocytes:
  - Normal range: 200-300 ms
  - Endocardium: ~279 ms
  - Midmyocardium: ~353 ms (longest)
  - Epicardium: ~280 ms

Species variation:
  - Small animals (mice/rats): <100 ms
  - Large animals/humans: 200-300 ms

Pathological conditions:
  - Long QT Syndrome (LQT1): 554 ms (vs 373 ms control)

TOTAL DEPOLARIZATION TIME:
--------------------------
  -  ~220 ms

6. ALIEV-PANFILOV MODEL PARAMETERS
================================================================================


PARAMETER SET 1 (Standard):
---------------------------
k        = 8.0     (cubic nonlinearity strength)
a        = 0.15    (threshold parameter)
epsilon0 = 0.002   (recovery time scale)
mu1      = 0.2     (recovery modulation)
mu2      = 0.3     (recovery modulation)

PHYSICAL MEANING:
-----------------
- k: Controls the steepness of the excitation threshold
- a: Determines the threshold for action potential initiation
- epsilon0: Base recovery time scale (smaller = slower recovery)
- mu1, mu2: Modulate recovery dynamics based on voltage and recovery state

MODEL CHARACTERISTICS:
----------------------
- Two-variable model: V (voltage), w (recovery)
- Modified FitzHugh-Nagumo model
- Accurately reproduces canine myocardium pulse propagation
- Simulates restitution properties with good precision


================================================================================
7. MEMBRANE AND CELLULAR PARAMETERS
================================================================================

MEMBRANE CAPACITANCE:
---------------------
C_m = 1.0 μF/cm² (standard value used in cardiac models)
  - Unit area capacitance of cell membrane
  - Relatively constant across cardiac cell types

RESTING POTENTIAL:
------------------
V_rest = -85 to -90 mV (typical ventricular myocyte)
  - Model often normalized to V_rest = 0 for computational convenience

PEAK DEPOLARIZATION:
--------------------
V_peak = +20 to +40 mV (during action potential upstroke)

TIME CONSTANTS (for relaxed monodomain model):
-----------------------------------------------
tau_v = 20 ms (relaxation time constant back to rest)
tau_w = 80 ms (recovery variable time constant)

These are phenomenological parameters for the simplified cable model used
in version2.py and version3.py, not biophysical measurements.

TEMPORAL RESOLUTION:

Simulation duration:

================================================================================
10. VALIDATION TARGETS
================================================================================

Your model should reproduce these key behaviors:

1. CONDUCTION VELOCITY:
   - Longitudinal: 400-600 mm/s
   - Transverse: 200-300 mm/s
   - Anisotropy ratio: ~2:1

2. ACTION POTENTIAL CHARACTERISTICS:
   - Upstroke duration: <5 ms
   - APD90: 200-300 ms (full Aliev-Panfilov model)
   - Refractory period: prevent immediate re-excitation

3. WAVE PROPAGATION:
   - Smooth elliptical wavefronts (due to anisotropy)
   - Major/minor axis ratio ≈ 2:1

4. INFARCT INTERACTIONS:
   - Wave blockage at infarct core
   - Slowing and bending at border zones
   - Potential for reentry around obstacles

5. STABILITY:
   - No spurious oscillations
   - Smooth repolarization
   - Consistent wavefront propagation
   - NEW no ring artifacts
